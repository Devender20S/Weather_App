# 📱 Weather App (Flutter)

A Flutter weather app that fetches real-time data from OpenWeatherMap API and displays current weather, hourly forecasts, and additional details.

## 🚀 Features
- Real-time weather data
- Hourly forecast
- Humidity, wind speed, pressure
- Clean UI

## 🛠️ Tech Stack
- Flutter (Dart)
- OpenWeatherMap API

## ⚙️ Setup
1. Clone the repo
2. Add your API key in `secrets.dart`
3. Run the app
